程式解析數位資料--以醫學數位內容為例

 

課程目標:

了解常見數位資料格式，熟悉某種臨床專業資料，看得懂資料處理程式範例，會修改範例程式進一步應用。

課程內容:

  1. 處理臨床資料的挑戰與機會
  2. C 程式分析資料
  3. FHIR 網頁表單
  4. 醫學影像
  5. 作業

LINE 社群連結 : https://line.me/R/ti/g/pLjuEMG9rN

資料範例

  1. 將臨床醫療量測的波形資料(如心電圖MIT ECG EBM ECG 轉換成標準規格，並於視窗或網頁介面中呈現
  2. 參考病理 virtual slide 系統(類似 google map) ，產生縮圖及截圖資訊。配合前端介面縮圖點選座標，呈現大圖部分內容
  3. 依據醫學影像呈現及標記程式範例醫學影像呈現及標記程式範例 ，取得標記座標資訊。設計使用介面，加入標記之解剖或病變說明資訊，轉換成醫資標準規格，上傳到 web 醫學影像平台
  4. 程式處理病變(或不同物種)之NGS 基因序列檢測資料。提供介面如 NGS IGV 範例 ，讓使用者瀏覽及選取特定基因片段，呈現其異變與參考基因的差異狀況(ATGC) 差異狀況
